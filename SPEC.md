# no golf! life! — 產品規格書

> 版本：v0.1 | 日期：2026-03-04 | 狀態：規劃中

---

## 一、產品定位

### 品牌名稱
**No Golf! No Life!**

### 一句話定位
> 給對高爾夫感興趣的新手，一個了解這個生活方式的入口。

### 定位類型
**生活風格策展媒體（Lifestyle Editorial）**
不是工具網站，不是學習平台，是有品味觀點的高爾夫入門導覽。

### 核心用戶
- 剛對高爾夫產生興趣，但覺得「很貴、很難、規矩很多、不知道從哪裡下手」
- 透過職場社交、朋友影響而開始想接觸高爾夫
- 不缺資訊，缺的是「有人幫我整理好一條路」

---

## 二、網站架構（Site Map）

```
no golf! life!
│
├── 首頁（Homepage）

├── 新手學習路徑（Learn）
├── 品牌地圖（Brands）
└── 頻道推薦（Channels）
```

---

## 三、各頁面規格

### 3-1. 首頁（Homepage）

**功能**
- 品牌第一印象
- 引導用戶進入四大功能

**設計方向**
- 參考：Brutus 日本雜誌風格
- 大字報式排版、強烈視覺層次
- 黑白為主 ＋ 重點色
- 像在翻雜誌的閱讀體驗

**內容元素**
- Hero 區塊：品牌 slogan ＋ 視覺
- 最新新聞預覽（3則）
- 學習路徑入口
- 品牌地圖入口
- 頻道推薦預覽

---

### 3-2. 新聞 / 情報（News）

**功能**
- 提供高爾夫最新資訊，培養用戶與高爾夫圈的連結感

**內容來源 Workflow**
```
AI 自動抓取高爾夫相關新聞
  → 人工審稿 ＋ 編輯
  → 發布到網站
```

> AI 抓取工具另外獨立開發，透過 API 或 CMS 串接發布

**文章元素**
- 標題、封面圖、日期、摘要、全文
- 標籤分類（賽事 / 球具 / 生活風格 / 規則）
- 相關文章推薦

**SEO 重點**
- 每篇文章獨立 URL（`/news/[slug]`）
- Meta title / description 完整填寫
- 結構化資料（Article schema）

---

### 3-3. 新手學習路徑（Learn）

**功能**
- 讓新手有清晰的入門路徑，知道「下一步要做什麼」
- 視覺上呈現旅程感（參考 Duolingo 地圖概念，但實作為網頁內容頁）

**學習階段（初版）**

| Stage | 主題 | 說明 |
|---|---|---|
| 1 | 遊戲規則說明 | 基本規則、禮儀、計分方式 |
| 2 | 姿勢說明 | 握桿、站姿、揮桿基礎 |
| 3 | 球場地形說明 | 球道、果嶺、沙坑等地形認識 |

> 後續可擴充：裝備入門、心理素質、第一次上場流程

**每個 Stage 包含**
- 文字說明（自製策展內容）
- 對應 YouTube 影片（嵌入播放 ＋ 縮圖 ＋ 頻道介紹）
- CTA：「準備好了嗎？找附近的練習場」→ 連結至地圖頁

**SEO 重點**
- 每個 Stage 獨立 URL（`/learn/stage-1`）
- 針對「高爾夫新手 怎麼開始」等關鍵字優化

---

### 3-4. 品牌地圖（Brands）

**功能**
- 讓新手快速了解高爾夫品牌生態，知道「買什麼、在哪買」

**架構：分類清單 ＋ 風格標籤**

```
類別（球桿 / 球袋 / 服飾 / 球鞋 / 配件）
└── 每個品牌
      ├── 品牌名稱 ＋ Logo
      ├── 風格標籤（例：#職人 #傳統 #巡迴賽御用）
      ├── 一句話介紹
      ├── 價格區間參考
      └── 適合新手？（是 / 否 / 進階）
```

**CTA 方向**
- 連結至品牌官網
- 未來可替換為聯盟行銷連結（不影響現有架構）

**SEO 重點**
- 品牌名稱 ＋「評價」「推薦」「新手適合嗎」等關鍵字
- 每個類別獨立 URL（`/brands/clubs`、`/brands/apparel`）

---

### 3-5. 頻道推薦（Channels）

**功能**
- 推薦高爾夫相關 YouTube 頻道，讓新手找到「學習偶像」

**每個頻道卡片包含**
- 頻道名稱 ＋ 頭像縮圖
- 一句話介紹（風格定位）
- 適合對象標籤（新手 / 中階 / 職業愛好者 / 生活風格）
- 代表影片嵌入（YouTube embed）
- 連結至 YouTube 頻道

## 四、設計系統方向

| 項目 | 方向 |
|---|---|
| 風格 | Brutus 日雜風、Editorial |
| 配色 | 黑白為主 ＋ 1個重點色（待定） |
| 字型 | 強烈粗體標題 ＋ 可讀性高的內文字型 |
| 排版 | 不對稱網格、大量留白、雜誌感 |
| 語氣 | 有觀點、不說教、帶點幽默感 |

---

## 六、SEO 策略

- 所有頁面完整填寫 Meta title / description
- 語意化 HTML 結構（h1 / h2 / article / section）
- 每篇內容獨立 URL（slug 為中文關鍵字對應英文）
- 圖片填寫 alt text
- 未來加入：Sitemap、robots.txt、結構化資料（JSON-LD）

---

## 七、CTA 策略（從第一天就埋）

- 每個 Stage 底部 → 導向下一個 Stage 或品牌地圖
- 品牌介紹 → 導向官網（未來換聯盟連結）
- 新聞文章 → 導向相關學習頁面
- 全站底部 → 「找附近練習場」（未來地圖功能）
